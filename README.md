# PDF Merger & Splitter

A simple web app to merge multiple PDFs or split one PDF into multiple files. All processing happens in your browser - no uploads, no servers.

## Quick Start

```bash
# Clone the repo
git clone https://github.com/JayMandaviyaDev/pdf-merger-splitter.git
cd pdf-merger-splitter

# Install dependencies
npm install
# or
pnpm install
# or
bun install

# Start the dev server
npm run dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Merge multiple PDFs with drag-to-reorder
- Split PDFs by page ranges or individual pages
- Client-side processing for privacy
- Progress tracking for large files
